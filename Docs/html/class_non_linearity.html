Detailed Description

This class provides mechanism for non linearity caluclation for provided 8x8 S-Box. Calculation are based on following document:

Generalised S-Box Nonlinearity
NES/DOC/UIB/WP5/020/A
Matthew G. Parker 1 2
Institute for Informatics,
University of Bergen, Norway

www.ii.uib.no/~matthew/SBoxLin.pdf
