Detailed Description

This class provides mechanism for generation irreducible polynomials. Used algorithm is based on the same concept as Eratostenes sieve. All polynomials with degree less or equal to provided max degree are considered, when algorithm starts. Polynomials, which are products of other polynomials are crossed out. After this operations only irreducible polynomials are left in the vector.
